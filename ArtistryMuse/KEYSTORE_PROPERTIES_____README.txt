The BuildItBigger App contains a file called "./keystore.properties" that is ignored by GitHub.

This file contains the cryptographic keys used to sign the App for use in the Google Play Store.

Replace the ellipses with the proper values.

----------------------------------
#
# These are the KeyStore values used to cryptograhically sign the APK before being submitting
# to the Google Play Store.
#

keyAlias=...
keyPassword=...
storeFile=...
storePassword=...
----------------------------------
